## 대부분의 사용자에게 권장되는 경로

대부분의 사용자는 웹 앱 저장소를 클론할 필요가 없습니다. 이미 PREQSTATION 배포에 접근할 수 있다면, 먼저 hosted 설정 경로를 사용하세요.

1. 웹 앱에 로그인합니다.
2. `/connections`를 열고 필요한 provider 설정을 완료합니다.
3. 원격 MCP 엔드포인트를 등록합니다.

```bash
# Claude Code
claude mcp add -s user --transport http preqstation https://<your-domain>/mcp

# Codex
codex mcp add preqstation --url https://<your-domain>/mcp

# Gemini CLI
gemini mcp add --scope user --transport http preqstation https://<your-domain>/mcp
```

4. 브라우저 OAuth 흐름이 뜨면 완료합니다.
5. 필요한 런타임에 맞춰 [워커 설치](/guide/ko/skill/installation) 또는 [OpenClaw 설치](/guide/ko/openclaw/setup)로 이어서 진행합니다.

API 토큰은 shell-helper 모드 또는 레거시 로컬 MCP 브리지를 써야 할 때만 나중에 필요합니다. 그 호환성 경로는 [API Keys](/guide/ko/web-app/api-keys)와 [Environment Variables](/guide/ko/reference/environment-variables)에서 확인하세요.

## Self-hosted 로컬 개발 사전 준비

아래 섹션은 self-hosting, migration 검증, 로컬 개발을 위해 PREQSTATION 웹 앱 체크아웃이 필요한 경우에만 사용하세요.

- **Node.js 22** 이상
- **PostgreSQL 13+** 이상
- [github.com/sonim1/preqstation](https://github.com/sonim1/preqstation)에 대한 **GitHub 접근 권한**

## 로컬 환경 변수

프로젝트 루트에 `.env.local` 파일을 만들고 `.env.example`을 복사한 뒤 로컬 개발에 필요한 값을 채우세요.

| 변수 | 필수 | 설명 |
| --- | --- | --- |
| `AUTH_SECRET` | 예 | HMAC 세션 서명 비밀값. 생성 예시: `openssl rand -base64 48` |
| `DATABASE_URL` | 예 | PostgreSQL 연결 문자열 |
| `ALLOWED_ORIGINS` | 아니오 | server action 및 same-origin 검사에서 추가로 신뢰할 origin |
| `UNSPLASH_ACCESS_KEY` | 아니오 | 프로젝트 배경 선택기에서 Unsplash 검색 기능 활성화 |
| `CRON_SECRET` | 아니오 | `/api/events/cleanup` cron 요청 인증 |
| `EVENT_OUTBOX_RETENTION_DAYS` | 아니오 | cleanup 작업 보관 기간, 기본값 `30` |

<Aside>
`drizzle-kit`은 `.env.local`의 `DATABASE_URL`을 읽습니다. 로컬 DB 명령을 실행하기 전에 `.env.local`을 준비하세요.
</Aside>

<Aside type="note">
표준 PostgreSQL 연결 문자열을 제공하는 관리형 서비스나 직접 운영하는 PostgreSQL이면 PREQSTATION과 함께 사용할 수 있습니다.
</Aside>

`PORT`는 런타임 env 스키마나 `.env.example`에는 없지만, 로컬 `npm run dev` 스크립트는 `.env.local`의 `PORT` 값을 읽어 기본 포트를 바꿀 수 있습니다.

## Self-hosted 로컬 개발 단계

### 1. 저장소 클론 및 설치

PREQSTATION 웹 앱 소스는 [github.com/sonim1/preqstation](https://github.com/sonim1/preqstation)에 있습니다.

```bash
git clone git@github.com:sonim1/preqstation.git
cd preqstation
npm install
```

### 2. 환경 변수 설정

```bash
cp .env.example .env.local
# PostgreSQL 연결 문자열과 필요한 선택 값을 포함해 .env.local 값을 수정합니다
```

### 3. 데이터베이스 마이그레이션

```bash
npm run db:migrate
```

이 명령은 저장소에 포함된 Drizzle 마이그레이션을 현재 데이터베이스에 적용합니다.

<Aside type="tip">
나중에 다른 PostgreSQL 서비스로 옮겨야 한다면 [데이터베이스 마이그레이션 가이드](/guide/ko/web-app/database-migration)를 참고하세요.
</Aside>

### 4. 개발 서버 시작

```bash
npm run dev
```

앱은 기본적으로 `http://localhost:3000`에서 열립니다. 다른 로컬 포트를 쓰려면 `.env.local`에 `PORT=`를 추가하세요. 로그인하지 않은 상태에서 `/`를 방문하면 `/login`으로 이동합니다.

## 첫 로그인

1. `http://localhost:3000/login`으로 이동합니다.
2. 새 데이터베이스처럼 아직 owner row가 없다면 브라우저의 설정 폼에서 첫 owner 계정을 생성합니다.
3. 이미 owner가 있다면 `users` 테이블에 저장된 이메일과 비밀번호로 로그인합니다.
4. 마이그레이션이나 기존 배포를 유지하는 경우에는 관련 데이터가 같은 owner ID를 계속 사용하도록 현재 owner row를 PostgreSQL에서 직접 업데이트하세요.

<Aside>
PREQSTATION은 single-owner 구조입니다. 모든 데이터는 PostgreSQL RLS를 통해 이 계정 기준으로 격리됩니다.
</Aside>

## Telegram 설정 (선택 사항)

OpenClaw와 Hermes를 모두 포함한 전체 흐름은 [Telegram 연동](/guide/ko/web-app/telegram) 문서를 보세요. 별도 chat ID, Hermes profile 설정, bot-to-bot 그룹 전달까지 그 문서에서 설명합니다.

최소한 **Settings > Telegram**에서는 아래를 저장하세요.

1. **Settings > Telegram**으로 이동합니다.
2. 공용 sender **Bot Token**을 입력합니다.
3. 일반 Telegram/OpenClaw 전달이 필요하면 **OpenClaw Chat ID**를 저장합니다.
4. **H Telegram** Hermes dispatch target을 쓰려면 **Hermes Chat ID**를 저장합니다.
5. 활성화한 채널에 대해 테스트 액션을 실행합니다.

토큰은 AES-GCM으로 암호화되어 저장됩니다.

## Shell Helper 모드 또는 레거시 로컬 브리지용 API 키

에이전트가 권장 원격 `/mcp` + OAuth 경로를 사용할 수 없고, shell-helper 모드 또는 레거시 로컬 브리지로 인증해야 할 때만 토큰을 발급하세요.

1. 로그인 후 `/api-keys`를 직접 열거나 워크스페이스 사이드바에서 **API Keys**를 클릭합니다.
2. **Create API Key** 폼에서 이름과 만료 기간을 입력합니다.
3. **Create Key**를 누른 뒤 성공 패널에서 토큰을 바로 복사합니다. 토큰은 `preq_`로 시작하며 한 번만 전체 값이 표시됩니다.

이 토큰은 에이전트 환경 변수 또는 레거시 `preqstation-skill` MCP 설정에 사용합니다.

```bash
export PREQSTATION_API_URL=http://localhost:3000
export PREQSTATION_TOKEN=preq_your_new_token
```

전체 호환성 설정은 [API Keys](/guide/ko/web-app/api-keys)와 [Environment Variables](/guide/ko/reference/environment-variables)에서 확인하세요.
