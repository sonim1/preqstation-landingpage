PREQSTATION은 Claude, Codex, Gemini를 실제 프로젝트에 연결하기 위한 실행 시스템입니다.

처음 설치하는 경우에는 아래 순서로 들어가면 가장 빠릅니다.

- [빠른 시작](/guide/ko/getting-started/quick-start) — 로컬 실행, 로그인, 첫 API 키 발급까지 한 번에 진행
- [웹 앱 설정](/guide/ko/web-app/setup) — `.env.local`, Drizzle 마이그레이션, 로그인 흐름 자세히 보기

구성은 세 가지입니다.

- **PREQSTATION 웹 앱** — 칸반 UI, REST API, Telegram relay, 실행 오버레이를 제공하는 중심 웹 앱
- **preqstation-skill** — 코딩 에이전트가 태스크를 읽고 상태를 갱신할 때 쓰는 MCP 서버와 shell helper
- **preqstation-openclaw** — OpenClaw 런타임에서 프로젝트를 찾고 worktree를 만들고 에이전트를 실행하는 스킬 패키지

이 시스템은 “이슈를 모아두는 백로그”보다, 이미 범위가 정해진 작업을 실제로 실행시키는 데 맞습니다.

태스크를 만드세요. 에이전트가 계획하고 구현하고 검증합니다. 마지막 판단과 승인만 당신이 합니다.
