PREQSTATION은 AI 에이전트 기반 개발 워크플로우를 위한 실행 플랫폼입니다. 개발자와 코딩 에이전트를 태스크 큐, 웹 앱, 에이전트용 API로 연결합니다.

## 세 가지 시스템

<CardGrid>
  <Card title="projects-manager" icon="setting">
    중심 웹 앱입니다. 태스크를 관리하고, 칸반 UI를 렌더링하고, REST API를 노출하고, Telegram/OpenClaw relay를 수행할 수 있습니다.
  </Card>
  <Card title="preqstation-openclaw" icon="rocket">
    실행 스킬 패키지입니다. OpenClaw가 이 패키지를 불러 프로젝트를 찾고 worktree를 만들고 코딩 에이전트를 실행합니다.
  </Card>
  <Card title="preqstation-skill" icon="approve">
    에이전트 클라이언트입니다. 코딩 에이전트가 실행 중 projects-manager API를 호출할 수 있게 해주는 MCP 서버와 shell helper입니다.
  </Card>
</CardGrid>

## 데이터 흐름

### 태스크 생성

```text
사용자가 칸반 UI에서 태스크를 생성
  -> projects-manager가 PostgreSQL에 저장
  -> 필요하면 Telegram / OpenClaw로 relay
```

### OpenClaw를 통한 실행

```text
개발자가 OpenClaw에 태스크 요청
  -> OpenClaw가 프로젝트 + 태스크 + 엔진 해석
  -> 격리된 git worktree 생성
  -> 요청된 코딩 에이전트 실행
```


## 기술 스택

| 레이어 | 기술 |
| ---------- | ------------------------------------------- |
| Framework  | Next.js 16 (App Router)                     |
| UI         | React 19, Mantine 8, Tabler Icons, Recharts |
| ORM        | Drizzle ORM                                 |
| Database   | Neon PostgreSQL (Row Level Security)        |
| Validation | Zod                                         |
| Auth       | Session cookies (HMAC-signed), bcryptjs     |

<Aside>
모든 데이터는 Neon PostgreSQL RLS를 통해 단일 owner 기준으로 격리됩니다. 현재 PREQSTATION은 개인 개발자 사용 흐름에 맞춰 설계되어 있습니다.
</Aside>
