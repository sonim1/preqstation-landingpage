## 사전 준비

- **Node.js 22** — [다운로드](https://nodejs.org)
- **PostgreSQL** — 클라우드 호스팅은 [neon.tech](https://neon.tech) 무료 플랜으로 시작할 수 있습니다
- **GitHub 접근 권한** — 공개 canonical 저장소는 `sonim1/preqstation`이지만, 로컬 웹 앱 설정은 아직 현재 private 개발 저장소 접근 권한이 필요합니다

## 설치

### 1. 저장소 클론

공개 PREQSTATION 저장소는 [github.com/sonim1/preqstation](https://github.com/sonim1/preqstation)입니다. 웹 앱 소스가 아직 그쪽으로 완전히 옮겨지지 않아, 로컬 설정은 현재 private 개발 저장소를 사용합니다.

```bash
git clone git@github.com:sonim1/project-manager.git projects-manager
cd projects-manager
```

아직 접근 권한이 없다면 계속 진행하기 전에 maintainer에게 GitHub 초대를 요청하세요.

### 2. 환경 변수 설정

`.env.example`을 `.env.local`로 복사한 뒤 필요한 값을 채우세요.

```bash
cp .env.example .env.local
```

그다음 `.env.local`을 수정합니다.

```bash
# Required
AUTH_SECRET=<openssl rand -base64 48 로 생성>
DATABASE_URL=postgresql://...

# Optional
PORT=3000
ALLOWED_ORIGINS=http://localhost:3000
```

소유자 계정은 환경 변수로 읽지 않습니다. 마이그레이션 후 PostgreSQL `users` 테이블에 owner 행을 넣으세요. SQL 예시는 [웹 앱 설정](/guide/ko/web-app/setup)을 참고하세요.

### 3. 의존성 설치

```bash
npm install
```

### 4. 데이터베이스 마이그레이션

```bash
npm run db:migrate
```

### 5. owner 계정 등록

마이그레이션 후 PostgreSQL에 owner 계정을 생성하거나 업데이트합니다. SQL 예시는 [웹 앱 설정](/guide/ko/web-app/setup)을 참고하세요.

### 6. 개발 서버 시작

```bash
npm run dev
```

앱은 `http://localhost:3000`에서 실행됩니다. 로그인하지 않은 상태에서 `/`를 열면 `/login`으로 리다이렉트됩니다. PostgreSQL에 저장한 owner 이메일과 비밀번호로 로그인하세요.

## 에이전트 연결

1. 로그인한 뒤 `/connections`를 엽니다.
2. Claude Code, Codex, Gemini CLI에 PREQSTATION 원격 `/mcp` 엔드포인트를 등록하고 브라우저 OAuth를 완료합니다.
3. 연결된 클라이언트와 만료 정보는 같은 페이지에서 확인합니다.
4. 셸 헬퍼가 필요하면 `/api-keys`가 `/connections?legacy=1`으로 이동합니다.

토큰은 셸 헬퍼 호환이나 직접 REST 자동화에만 사용합니다.

```bash
export PREQSTATION_API_URL=http://localhost:3000
export PREQSTATION_TOKEN=preq_your_new_token
```

이제 에이전트 클라이언트 또는 MCP 설정으로 넘어가면 됩니다.

## 실행 경로 선택

PREQSTATION에서는 worker와 dispatcher를 따로 봅니다.

1. 먼저 [워커 설치](/guide/ko/skill/installation)를 진행합니다.
2. 그다음 필요하면 [OpenClaw 설치](/guide/ko/openclaw/setup)를 진행해 `Send to Telegram`, Telegram/OpenClaw 명령, queued QA dispatch를 처리합니다.
3. 웹 앱의 queued dispatch가 필요 없다면 worker 설정만 하고 끝내면 됩니다.
