PREQSTATION is a task management platform designed for AI-agent-driven development workflows. It bridges developers and AI agents through a central task queue and REST API.

## Three-System Architecture

PREQSTATION consists of three interconnected services:

<CardGrid>
	<Card title="project-manager" icon="setting">
		Central web service. Manages tasks, renders the Kanban UI, exposes the REST API, sends Telegram notifications.
	</Card>
	<Card title="preqstation-openclaw" icon="rocket">
		Trigger layer. Receives Telegram messages and launches coding agents in isolated git worktrees.
	</Card>
	<Card title="preqstation-skill" icon="approve">
		Agent client. MCP server and shell helpers for agents to call project-manager APIs during execution.
	</Card>
</CardGrid>

## Data Flow

### Task Creation and Notification

```
User creates task in Kanban UI
  → project-manager stores in PostgreSQL
  → (optional) sends Telegram notification
```

### Agent Trigger via Telegram + OpenClaw

```
Developer sends task request via Telegram
  → OpenClaw receives message
    → preqstation-openclaw skill triggers
      → Parses: engine, task ID, project key
      → Resolves project path from MEMORY.md
      → Creates isolated git worktree
      → Launches coding agent (claude/codex/gemini CLI)
```

### Agent Execution and Status Reporting

```
Coding agent fetches task via preq_get_task, then:

  inbox       → preq_plan_task (write plan, → todo)
  todo        → preq_start_task (→ in_progress)
              → Implement + test + deploy per strategy
              → preq_complete_task (→ review)
  in_progress → Continue implement + test
              → preq_complete_task (→ review)
  review      → Run verification (tests, build, lint)
              → preq_review_task (→ done)

  On any failure → preq_block_task (→ blocked)
```

## Tech Stack

| Layer      | Technology                                  |
| ---------- | ------------------------------------------- |
| Framework  | Next.js 16 (App Router)                     |
| UI         | React 19, Mantine 8, Tabler Icons, Recharts |
| ORM        | Prisma 6                                    |
| Database   | Neon PostgreSQL (Row Level Security)        |
| Validation | Zod                                         |
| Auth       | Session cookies (HMAC-signed), bcryptjs     |

<Aside>
All data is owner-scoped via Neon PostgreSQL Row Level Security. PREQSTATION is designed for single-developer use.
</Aside>
