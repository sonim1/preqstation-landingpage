PREQSTATION is an execution system for developers using Claude, Codex, and Gemini in real projects.

It consists of three interconnected systems:

- **PREQSTATION web app** — Central web app with the Kanban board, REST API, Telegram delivery, and execution overlays
- **preqstation-skill** — MCP server and shell helpers that coding agents use to read and update tasks
- **preqstation-openclaw** — Skill package for OpenClaw runtimes that resolves projects, creates worktrees, and launches agents

Use it when a task is already scoped and you want reliable execution, not when you need a generic issue backlog.

Create tasks. Agents plan, implement, and verify. You review the result and decide what reaches `done`.
