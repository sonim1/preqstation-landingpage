PREQSTATION is a task management platform designed for developers who want AI agents to work on their behalf.

It consists of three interconnected systems:

- **project-manager** — Central web service with Kanban board, REST API, and Telegram notifications
- **preqstation-skill** — MCP server that coding agents use to read and update tasks
- **preqstation-openclaw** — Trigger layer that launches agents from Telegram messages

Create tasks. AI prepares them. You review the results.
