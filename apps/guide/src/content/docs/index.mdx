PREQSTATION is a task management platform designed for developers who want AI agents to work on their behalf.

It consists of three interconnected systems:

- **projects-manager** — Central web app with Kanban board, REST API, Telegram delivery, and execution overlays
- **preqstation-skill** — MCP server and shell helpers that coding agents use to read and update tasks
- **preqstation-openclaw** — Skill package for OpenClaw runtimes that resolves projects, creates worktrees, and launches agents

Create tasks. Agents plan, implement, and verify. You review the results.
