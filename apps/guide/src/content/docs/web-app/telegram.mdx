Telegram is optional in PREQSTATION. The current app uses it for explicit `Send to Telegram` actions and the legacy OpenClaw relay path, not as a full automatic notification bus for every task event.

## Setup

### 1. Create a Telegram bot

1. Message [@BotFather](https://t.me/BotFather)
2. Run `/newbot`
3. Complete bot creation
4. Copy the bot token

### 2. Configure it in PREQSTATION

1. Open **Settings > Telegram**
2. Paste the bot token
3. Add your chat ID
4. Click **Test Connection**

### 3. Confirm the chat ID

The test message reveals the chat ID that PREQSTATION will use for outgoing messages and relays.

## Token Security

Telegram bot tokens are encrypted at rest using AES-GCM with a key derived from `AUTH_SECRET`.

Stored format:

```text
v1.{base64urlIV}.{base64urlCiphertext}
```

## What Telegram Is Used For

Telegram is best treated as a delivery channel for:

- explicit task messages sent from the UI through **Send to Telegram**
- legacy OpenClaw relay messages

## Claude Code dispatch does not require Telegram

The PREQSTATION web app now also supports a separate **Send to Claude Code** path. That route queues `dispatch_target=claude-code-channel` and is handled by the local Claude dispatch runtime in `preqstation-skill`.

Use Telegram only when you want the OpenClaw relay path.

### Disable notifications

1. Open **Settings > Telegram**
2. Turn off **Enable notifications**

## Relaying Requests to OpenClaw

Telegram can also carry OpenClaw requests. Prefer natural language or an explicit skill call rather than the old fixed `CLAUDE PROJ-123` format.

Examples:

```text
Use Codex to implement PROJ-123 in the example project.
Start PROJ-123 in sparse mode using Claude Code.
!/skill preqstation-dispatch review PROJ-123 using gemini-cli
```

The OpenClaw runtime receives the message, resolves the target project, creates a worktree, and launches the requested agent.

<Aside>
If your Telegram bot token is exposed, revoke it in [@BotFather](https://t.me/BotFather) and save a new token in PREQSTATION.
</Aside>
